About

Tibis (tibis the IBIS) is a tool to help manage encrypted directories (named data container here).

Name is inspired by the Egyptian god of writers and protector of knowledge.

Basically Tibis is GPG and TAR put together with a command line interface and configuration file.

Many other tools may provide the same functionnalities like VeraCrypt or Zed, however, here, the code is simple, and the technology is known.

How does it works ?

Create a new data container
tibis create < name >
tibis will ask you a passphrase, choose a good one !

List data container
tibis list

Mount a data container
tibis unlock < name > < destination path >
tibis will ask your passphrase.

Lock a data container
tibis lock < name >
tibis will not ask your passphrase.. because it's PGP Public key.

Remove an entry
tibis delete < name >
tibis will ask you a confirmation, take care !

Why ?
Because we have to handle with sensitive files, and store them securly can be boring

How ?
Tibis is a kind of python wrapper on pgp keys and gzip files. Yes, nothing more.

Why do we have to set passphrase ?
Security is a matter of trust, if you don't trust your laptop enough to store sensitive data, why leave the keys in it ? By using a passphrase, access to the data needs : something you have ( data / key ) and something you know ( passphrase )
